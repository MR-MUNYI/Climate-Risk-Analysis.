# Climate-Risk-Analysis.
# Climate Risk analysis and Rainfall prediction using Python.

## Project Overview
This project analyzes climate data including temperature and precipitation
to identify climate patterns and potential climate risks.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset Variables
- Period
- Month
- Mean Temperature
- Max Temperature
- Min Temperature
- Precipitation

## Project Steps
1. Data cleaning
2. Exploratory data analysis
3. Visualization of climate trends
4. Rainfall prediction using machine learning

## Applications
- Agriculture planning
- Climate monitoring
- Insurance risk analysis
